# excel2json
Convert Excel to JSON

## Usage
```
NAME:
   excel2json - Convert Excel to JSON

USAGE:
   excel2json [global options] command [command options] [arguments...]

VERSION:
   0.0.1

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --file value, -f value   excel file path
   --sheet value, -s value  parse excel sheet
   --help, -h               show help (default: false)
   --version, -v            print the version (default: false)
```
